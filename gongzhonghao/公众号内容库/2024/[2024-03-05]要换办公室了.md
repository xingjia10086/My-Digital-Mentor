**  
** ** 这是星佳和爱米的第  1237  ** ** 篇原创  ** **持续记录从宁夏到深圳和香港的孟母三迁之路**

  

  

年后一直很忙，本想着安顿好团队后继续开拓新业务，于是出去学习了一阵子，也拜访了一些前辈，想都是问题，做才会有答案。

  

回到公司被写字楼物业通知，因为房东出了点问题，这层写字楼已经被政府强制执行过户，需要清场之后公开招标租客或者打包拍卖，于是给我们这一层的租客们提前通个气，建议我们提前开始找新办公地。

  

万万没想到，创业的旅途中，先没了的居然是房东。

  

算了下最近新入职的同事们人数，还有结合目前孟母三迁每天的实际情况，我们需要有至少4个单独的房间供顾问老师和来自全国各地的父母们面对面的沟通，而外面的工位按照目前的规模，起码要20个才勉强够用。  

  

享受了这么久世贸的物业服务，还有周边的地铁，商圈，书店和餐厅等等的配套，肯定要换个更上一层楼的，于是联系了几个中介提了明确的需求去找。  

  

现在的写字楼中介比二手房中介的确专业很多，需求一提明确，人家很快用笔记图文并茂还有视频演示，各种必需信息一应俱全，很为客户节省时间，这就是市场经济的魅力。

  

这两天同事们是真的开始卷起来了，很明显觉得战斗力爆棚，人的确是很容易被环境所影响的，尤其是一群为了自己为了孩子也为了家庭的父母们。  

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqvRfZ6ngYzYY6qDzGA9uEWfV0DUDpmQoj1q5ibs41zFAOn87YLMx3pCmicQST0CNuNdggOzdzk13B3Q/640?wx_fmt=png&from=appmsg)

  

最近不知道啥原因来应聘的越来越多，凡是想做顾问老师的，我都会问三个问题：  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

  1. 有没有成家？  ‍  ‍  ‍  ‍  ‍ 

  2. 有没有孩子？ 

  3. 有没有香港身份？ 

  

按照孟母三迁对于顾问老师的要求，市面上能符合我们要求的同事是真不多，但就是因为有如此严格的要求，才能确保每个来孟母三迁的同事们想要的不仅仅是这一份工作而已，更重要的是未来的发展，和家庭的规划，我们深刻理解一件事，孩子能培养成什么样子，完全取决于父母给孩子们做了什么榜样。
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

而家庭建设里，最重要的无非是夫妻关系，尤其是家族里的那个女人。  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

每过一阵子，互联网上就会把河南父母高考移民的话题上热搜，经过和很多郑州父母们沟通我才真的理解他们的不容易，高考地狱的确实至名归，于是很多父母会去天津，西安，海南，兰州或者宁夏去寻求换个赛道，然后代价是要买房，要落户，要初二就去陪读。
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

其实掐指一算，这些成本和香港比，到底是消费还是投资呢？  ‍  ‍  ‍  ‍

  

  

河南家长之所以认知高，执行力强，完全是因为早早就开始研究内地高考移民了，最后发现还是香港性价比最高。  

  

今天大家在群里写咨询日志，看到深国交妈妈倩姐写的一段话：  ‍  ‍

  

> 客户大女儿4岁，计划生二胎，办理香港身份主要是为了孩子享受更优质的教育资源 李女士身边有不少朋友办理了香港身份，了解到移居香港对子女有不少优势。
>
>  
>
>
> 一方面：对于孕妈来说，在香港可以享受先进的医疗技术和服务。
> 另一方面：对于孩子来说，在香港出生的中国公民，即为香港永久性居民，可以享受到香港包括教育、医疗在内的全部福利。
>
>  
>
>
> 因为此前李女士的朋友通过孟母三迁成功办理了香港身份，所以李女士直接联系了我们。
>
>  
>
>
> 经过初步评估，李女士有百强大学本科学位，且工作经历超过十年，符合申请香港高才B类的要求。
>
>  
>
>
> 在李女士确认此方案后，准备让孟母三迁开始全程协助办理。

  

还有我们二胎妈妈的东北马姐，来了深圳快二十年，和先生成为了我们同事嘴里的榜样夫妻，在选择为孩子买学区房还是办香港身份之间，选择了全都要，不仅买深圳的学区房，拿到身份之后再买一套香港的自住房。

  

这才是孟母三迁的精神，只要父母能解决的，绝对不麻烦孩子们！  ‍  ‍  ‍  ‍  ‍

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqvRfZ6ngYzYY6qDzGA9uEWfLWUctZWosSHff6g3D1r9aIv2UHtoYMSXhadHx9eX6d2EMu4ia5Wmf2A/640?wx_fmt=png&from=appmsg)

  

下午来了一对父母，很特别，因为是小星星同学的爸妈，星佳和爱米不会在家长群里宣传这些，但是朋友圈和公众号的确是坚持写，有时候不得不怀疑，我们以为的常识，的确是很多人至今都还不知道的，哪怕是他们已经来了深圳快十年。
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

好在高学历的父母们还有学习能力，虽然是通过我们知道了香港这条赛道，但是人家能花心思真的去研究这件事是否靠谱，于是这是第二次来找我们落实一些细节后成功签约。
‍  ‍  ‍  ‍  ‍

  

把这些记录下来之后，或许等以后小星星懂事的时候翻出来会看看，才能理解什么叫父爱如山吧，母爱如水吧，晚上和同学爸爸聊聊，回想18年一穷二白到深圳，的确挺不容易的。
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

疫情期间有了小太阳，然后爱总去香港读研，的确也挺不容易的。  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

做孟母三迁起初就我们三个人，如今已经20多人，又要换办公室，一眨眼已经过了三年多了，的确也挺不容易的。  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍
‍  ‍  ‍  ‍

  

经历了这么多不容易，下一个目标就是去香港生活，买房和创业，会觉得其实也没那么难吧。  ‍  ‍  ‍

  

  

  

但这些不容易本身就是很难得的人生经历和体验，让我们过的每天都不一样，觉得每天都是为了自己活，真好！  ‍  ‍
