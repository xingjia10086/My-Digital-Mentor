最近的娱乐消遣之看书和游戏  ‍  ‍  ‍

  

  

虽然Zlib复活很久了，但不可否认的是微信读书才是最强的，配合电纸书的体验，真的很容易让我们在零碎时间打开书架翻翻，比如你可以订阅公众号，然后一口气就可以看完。

据说得到电纸书做的也不错，希望啥时候可以出个墨水屏版就好。  ‍  ‍  ‍  ‍

其实并不是所有女人都对综艺和八卦感兴趣的，就跟影响孩子一样，平常你经常看书，写作和运动，孩子自然会跟着你有模有样的学，这就是镜像效应。  ‍  ‍  ‍

  

而启发家里的另一半，就从一起看优质的纪录片和访谈开始，最好的学习就是 提个好问题，然后一起探讨，听听不同角度的答案。  ‍  ‍  ‍  ‍  ‍  ‍

当然，这种鸡汤也不能天天喝，人终究是很容易被环境影响的，比如这本书，当个故事看还是很启发斗志的，幸福终究是一种对比而来的感受。  ‍  ‍  ‍  ‍  ‍

  

昨天看俞敏洪对于物美集团创始人张文中的采访，虽然肯定删减了很多，但短短40多分钟的对话，让星佳很佩服这两个快奔60的中国第一代企业家，他们的人生故事编剧都写不出来。

在公众号里其实写了蛮多的读书笔记和观后感，每过几年伴随着人生经历，同一本书又能读出来不同的味道，所以人生的意义就是在于去寻找人生的意义吧。  ‍  ‍  ‍
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

除了看书，去看电影也是必须的，不知道为何如此多的人会对《超能一家人》差评，当然我也不过看完电影后好奇打开豆瓣看看别人能有什么观后感，的确跟看书一样，同一本书籍，不同的人能看出来不同的东西，进而会有不同的行动，有些人骂骂咧咧说被骗了40块看了烂片（还能得出剧组没钱雇佣水军等等不知道哪里来的是非），有些人却更珍惜和家人在一起的时光。
‍  ‍  ‍  ‍  ‍

  

只要交易是自由且平等，不是被迫的，那就会增加财富，艺术和消遣本来就是和情绪价值的交易。  ‍  ‍  ‍  ‍  ‍

  

微信读书的算法的确挺厉害的，总能推送给星佳想看的书，有些书适合一字一句的读，有些书一目十行脑袋里有个印象就挺好，所以也没必要去纠结是不是必须买实体书，毕竟互联网时代如何获取有效信息才是衡量一个人能力的关键，而不是靠花钱就能随便买来的东西。
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

其实一直都没有沉迷过游戏，但得益于英伟达公司老黄的传奇故事，最近会花点时间体验下某些游戏，比如战地系列，It takes

临在时刻

—  

  

  

今天和孟母三迁的小伙伴去考察DSE的线下培训机构，到场的二十多个家长里，很多都是高二或者中考完的孩子，问他们从哪里知道DSE的，他们都说因为的确孩子国内高考没戏，只能想办法，如果DSE还搞不定，只能砸钱送出国了。  
  
即使是深圳湾的家长，越来越多的居然把送孩子出国留学当成了最后的选择，他们知道星佳和爱米孩子才6岁就已经来研究升学都很羡慕我们能早早规划。  ‍  ‍  ‍
‍  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqsChIXIMqEPBeYk8rtQsYB0Snv0EB99GhK7z7VnxfMo4WHZ0PMOE3b9PwqpCeKiaXFhYHia48ZMM4ibg/640?wx_fmt=png)

  
但说真的，这些年一直有个思想是「想去哪里，就从那里开始」，既然家庭的大事无非是选对城市，买对房子，找对另一半，提前多生孩子和考个好大学，那不如趁早研究清楚AP，DSE，OSSD，SAT等等不同的升学路径到底是什么，知己知彼才能运筹帷幄啊。
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  
可惜的是，参加了这么多场线下讲座，大部分都是妈妈带着孩子，爸爸很少出现，回家路上还在和爱总探讨，这些家庭大事不是应该父母一起去学习，一起去跟专家聊，然后一起做决策的吗？
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  
如此简单且朴素的道理，好像能知行合一的父母真不多，包括最近接到的蛮多付费咨询里，不止买房还是香港身份，很多都是家里掌握财权（所谓对外的男人）说了算，于是直接否定了妈妈经过深思熟虑后的参考意见。
‍  ‍  ‍  ‍  ‍  ‍  ‍  
除了，上面随笔写的几个运营家庭的人生关键节点，早点开始考虑自己的养老规划吧。

  

建议大家35岁之前就开始考虑父母，和自己的养老，因为养老这件事只能靠自己未雨绸缪了。
