儿童节的礼物

  

  

今天小星星的第六个儿童节，也是小太阳的第二个儿童节，那肯定要写点什么，太煽情的星佳也搞不来，但翻翻公众号的回忆还是值得的。

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqtibTbCicLkncbJzElVadU8V1zIJtqh351BiadfLytyQmDHjnD7gVE8yUKVVqfI521IS2vsDweKEzqeA/640?wx_fmt=png)

  

总不能像爱总这样，明明说好每天坚持直播一小时的，结果此刻用儿童节当礼物，正点了宵夜在一旁大快朵颐呢，所以说，真的能坚持一件小事才是跟时间做朋友，比如基本上每天坚持写一篇上千字公众号文章的星佳。  

  

提起儿童节，搜索了下公众号的历史记录，第一篇居然是15年的61写的（ 那时候某个人还是未婚妻的角色 ）：  

  

>
> 电影院里大多数都是父母带着孩子一起来看的，前半段此起彼伏的笑声，不知道那些小小年纪的孩子们能不能懂男女主角之间纯纯的爱，尤其最后那段女主角结婚前夜跟老爸阳台上的对话真的是针针见血啊。
>
>  
>
>
>
> 想起来日本婚姻后女方就得呆家里相夫教子，男人跟小新爸爸一样房贷车贷跟巨大的生活压力也是头疼，不过生活对于我们来说有时候就是一个巨大的玩笑，往往不幸之中挺过来后才会体会真实的生活滋味，所以听那些过来人讲的大道理都会觉得真由道理啊。
>
>  
>
>
> 记得之前看过一句话，电影就是应该让人们看到生活里看不到的美好，因为生活中已经够惨了。
>
> 公众号：星佳是个小人物 [ 每个人心里都有个哆啦A梦....
> ](https://mp.weixin.qq.com/s/xSNxkpbK8PAzKQZMpwXfrA)

  

过了这么多年，翻出来这篇文章还觉得挺有趣，其中有一段关于星佳父母的描写：  

  

> 看完电影收到了老爸发来的儿童节快乐，潮爹还用美图秀秀拼了个小时候星佳的图，看起来满满的回忆啊。
>
>  
>
>
> 说真的，老爹貌似就是星佳身边的哆啦A梦啊，不知道你对于父母的情感是不是也是这样，总是无条件不求回报的满足不懂事的自己的各种需求，哪怕是能力范围外的强求。
>
>  
>
>
>
> 星爹在星佳上学的那么几年里每天早晨会坚持做各种早餐，现在换个角度看自己，老爹也真是满满爱啊，不知道以后星佳当了爹会不会这样为小星佳付出，看看现在如此的心智健康跟正常的人格真的要多谢父母的培育啊，想当年想混社会走歪人生路也是在一念之间的事情。
>
>  
>
>
> 所以到了这个年纪就越来越懂当初父母的安排跟嘱咐，虽然偶尔还是会任性的跟父母吵吵闹闹，不过让这就是爱吧？
>
> 公众号：星佳是个小人物 [ 每个人心里都有个哆啦A梦....
> ](https://mp.weixin.qq.com/s/xSNxkpbK8PAzKQZMpwXfrA)

  

![](https://mmbiz.qpic.cn/mmbiz/tOFSAibJ5wquMBWYIQLiax6asGcu4I5IH1NW8ict8R5usYOh19occzErs1uQWYB0T7v8DBMnkoIkW6iak8HkicQv9ag/640?wx_fmt=jpeg&tp=wxpic&wxfrom=5&wx_lazy=1&wx_co=1)

  

自从也为人父母后，才真的理解当年自己父母的真心啊。  

  

之后是21年的61儿童节写的《 [ 深吹一家人的腾讯直播首秀
](https://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649815285&idx=1&sn=c2a79748a48903b8d037b25017af0bbd&scene=21#wechat_redirect)
》，原来星佳那么早就开始时髦的玩直播了，而且如今看这篇文章，真是更多感触啊！  

  

>
> 讲了一句话「笨拙的人讲道理，聪明的人讲故事」，别说家人，有时候说服自己都挺难的，而自信这个东西其实是很多次的小成就慢慢积累起来的大勇气吧，跟让家人同意也是一个道理，毕竟他们也是怕你太辛苦，打心眼里是希望你们能过的好，所以只要你做出来的事情（或者讲的故事）能够让他们放心，其实家人何尝不让你出去到大城市试试看看更高层的风景呢？
>
>  
>
>
>
> 比如你会不会把你认为好的文章发给爸妈看看，能不能留点时间跟他们探讨他们不懂的一些问题，坐下来好好的聊聊未来，没有哪个亲身父母真的乐意把你圈在身边关在牢笼里的。
>
> 星佳，公众号：星佳是个小人物 [ 资深深吹一家人的腾讯直播首秀，2200+人气在线观看的有趣体验....
> ](https://mp.weixin.qq.com/s/jYpX8vfUtFStR5crRbnWPw)

  

扫了文章里的回放二维码，时隔两年多，腾讯直播这个服务居然已经停止维护了，互联网时代的变化真快啊。

  

之后是21年的儿童节写的《 [ 一份特殊的儿童节礼物
](https://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649817904&idx=1&sn=cd8283510bc8c59431a8d3645107738b&scene=21#wechat_redirect)
》，那时候还自称软饭星佳，我们一起去幼儿园参加小星星四岁的儿童节表演，然而当天迎来的礼物是国家支持三胎的政策。  

  

那篇文章后的留言如今看自己还真低调（ 那时候其实爱总已经大肚子一阵子了
），如今我们不仅已经有了快两岁的小太阳，还因为原本计划把他生香港而给爱总折腾了研究生，香港身份+上海户口等等福利。

  

当然最大的收获是跟一群志同道合的小伙伴们经营着如今践行6个早点和国际化的 孟母三迁 小公司。  

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqtibTbCicLkncbJzElVadU8V1NhuzibkQa5yj4FHerjuYAZqcMdSlssibw41e0RMLQBzNjlPSfcrhYOwg/640?wx_fmt=png)

  

>
> 星佳一直坚信孩子就是上天派来拯救自己的天使，如果不是在27岁的时候有了小星星，真不敢想象如今会过的什么内心煎熬又郁郁不得志的苦日子，是不是还在小地方整天抱怨运气不好呢？
>
>  
>
>
> 对于敢于负责任的父母来说，孩子正是他们的动力，因为他们的成长不会等你的焦虑和迷茫散去，唯有实干才能解决问题。
>
>  
>
>
> 去年的父亲节，写了篇文章《
> 第三个父亲节，终于把给孩子吹的牛实现啦！》，其实跟夫妻关系一样，我们跟孩子也是彼此成就的，至少如今星佳和爱米的理想生活里少不了孩子，而小星星也开始期待可以成为一个靠谱的大哥，保护弟弟或者妹妹，就跟
> 快把她哥带走 里的主角那样。
>
> 软饭星佳，公众号：星佳是个小人物 [ 一份特殊的儿童节礼物
> ](https://mp.weixin.qq.com/s/zQXQLrUVAKitGqJmgiDSrQ)

  

觉得孩‮来子‬到这‮世个‬间，之所‮是以‬父母的最大幸运，原因就在这‮一里‬的出生，事实上在逼‮着迫‬你重新启动学习，甚至‮新重‬做人。

  

难道你‮人为‬父母之后‮有没‬这样的感受吗？

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqtibTbCicLkncbJzElVadU8V1JYglrR1RmluibK4X99UpVunlTB8yb76sjX3WEq7azVeQXtd1McV849w/640?wx_fmt=png)

  

而22年写的儿童节文章《 [ 孟母三迁喜报分享
](https://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649823232&idx=3&sn=0fccf53e65c273fe65bd417f979fb508&scene=21#wechat_redirect)
》，是因为帮客户拿到了理想中的Offer，于是客户在工作群里的实名感谢，慢慢从解决自己家庭的小问题转化到成就其他家庭的孟母三迁事业。  

  

今天从世界之窗的萌宠俱乐部开车回家的路上，老妈说当年的小星星也是如今小太阳差不多大的时候，那时候我们还住龙华，有一次带小星星去世界之窗玩，结果臭小子又哭又闹，因为看到了一个人的背影认成了妈妈，而那天我也记得清楚，还在腾讯搬砖压根走不开。  

  

好在过了这些年，我们终于可以在哪怕不是休息日的六一儿童节陪孩子去自由的玩，甚至也有能力带他去外面的世界再好好看看，香港，日本，美国，加拿大，爱尔兰等等。

  

那就给下一个儿童节定个愿望清单吧！
