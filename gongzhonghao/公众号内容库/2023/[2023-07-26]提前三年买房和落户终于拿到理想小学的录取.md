为了去理想的小学  ‍  ‍  ‍  ‍

需要提前三年开始筹备

  

  

今天下午 3 点，南山区幼升小的成绩终于公布，小星星如愿拿到了理想中的小学录取。

  

按照去年该小学的录取分数线，其实我们多多少少还有点风险，简单点说，我们买房的时间和孩子落户的时间跟很多同样报名的家长们比，真没优势，毕竟一堆孩子刚出生就把户口落在自己家买的学区房了，所谓赢在怀孕前!

  

好在，深圳总是可以让一些人心想事成的，规则所有人都有权利去查询，按照规则去实现自己的目的就好，找到理想的小学，然后按照规则去争取积分够高就好，嗯，很效率很公平。
‍  ‍  ‍  ‍

  

对于小星星的小学如何选择，我们优先考虑的是距离家近，方便接送，并且不希望小学就花太多精力去刷题和死记硬背，一定要保证孩子的睡眠和运动，小学只需要培养学习和生活习惯就好，所以父母能想清楚对于理想学校的画像是什么很重要。
‍  ‍  ‍  ‍  ‍  ‍  ‍

  

那天和爱总聊，如果真的没被录取，我们也没打算去小区周边的其他学校，只能带小星星去香港或者日本继续读书了，正所谓孟母三迁的业务都是被孩子助推起来的，一切都是最好的安排也挺好。
‍  ‍  ‍  ‍

  

前两天写了两篇文章在普及自己对于小星星未来教育规划的思考，看到惨淡的阅读量其实挺开心的，先知先觉的人起初总是不会被大众所理解，这是好事，正如国内目前能知道
DSE 是什么的父母或许还不到 2% 的群体，不然就没那么多人卷不动高考宁可去卷中考了。  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍
‍  ‍  ‍  ‍  ‍  ‍

中午和客户吃饭，这是第一次跟他见面，居然都要去香港拿身份证了，孩子马上高二，自然很关心对于孩子的安排，他提到一个很不错的观点，到底应该如何建议孩子选择本科的专业？
‍  ‍

  

看看中美之间彼此限制发展的行业是什么，比如半导体，AI，大数据，生物制药，航天等等，尖端行业其实就是人才之争，而人才的竞争是实实在在要花钱才能换来的，这就是战略勤奋比战术更重要的案例。
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqv7vkVzldXtu5CDkA2TFDKR1szo6o3hMX6O4RhgvjZlZnxI5qZ5HLBq4wyWUfLDXo6xjCgOs0bHng/640?wx_fmt=png)

  

这个客户做的是一带一路国家的贸易生意，上个月刚从俄罗斯回来，于是一旁的我们很好奇到底那边战争的情况如何，但真的听当事人说才知道，我们在媒体上看到的画面和新闻其实都不是事实，相反他的业务有了巨大增长，今年借着香港的国际化要把那边的工厂规划再增加很多。
‍

  

很多时候，与其说孟母三迁为客户服务，不如说更占便宜的是我们，每次跟他们请教，尤其是他们做出结果，亲身实践的分享，给了我们很多思考，得到优质信息的能力是很稀缺的。

  

除了最近热闹的回流生事件，昨天宁夏的中考分数线公布后也引发了很堵讨论，从 08
年星佳高考那一年开始，其实就有大量的山东考生高三来宁夏参考高考，以至于那一年的一本线高达 498 分。

  

所以到了大学才知道跟同宿舍的同学比，宁夏的教育的确是落后，人家的高考分居然比我高了可能一百分，但我们上的是同一个大学的同一个专业。  ‍  ‍  ‍  ‍
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍

  

想起来河南的客户给我们讲了个身边朋友的段子，他的孩子从河南到西安开始高一，结果一路都是学霸，但发现河南的孩子和西安的孩子都很不适应。  ‍  ‍  ‍  ‍
‍  ‍

  

周六周末突然没有了补课，河南的孩子们一下子不习惯放松的生活了，但是西安的孩子本来周六周末就是放松去玩，突然间不得不要去适应补课了，于是两边的孩子有了冲突，开始了彼此都无法理解彼此的竞争，换句话说，文化冲突吧。

但说真的，这些年一直有个思想是「想去哪里，就从那里开始」，既然家庭的大事无非是选对城市，买对房子，找对另一半，提前多生孩子和考个好大学，那不如趁早研究清楚AP，DSE，OSSD，SAT等等不同的升学路径到底是什么，知己知彼才能运筹帷幄啊。
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  
可惜的是，参加了这么多场线下讲座，大部分都是妈妈带着孩子，爸爸很少出现，回家路上还在和爱总探讨，这些家庭大事不是应该父母一起去学习，一起去跟专家聊，然后一起做决策的吗？
‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  ‍  
如此简单且朴素的道理，好像能知行合一的父母真不多，包括最近接到的蛮多付费咨询里，不止买房还是香港身份，很多都是家里掌握财权（所谓对外的男人）说了算，于是直接否定了妈妈经过深思熟虑后的参考意见。
‍  ‍  ‍  ‍  ‍  ‍  ‍  
除了，上面随笔写的几个运营家庭的人生关键节点，早点开始考虑自己的养老规划吧。

  

建议大家35岁之前就开始考虑父母，和自己的养老，因为养老这件事只能靠自己未雨绸缪了。
