#  95后，才是我们这些老家伙的榜样...

原创  星佳  [ 星佳是个小人物 ](javascript:void\(0\);)

__ _ _ _ _

因为工作的关系，会跟95后接触的多一点，越跟他们混的久，就越觉得我们这些老家伙是真的落伍了，不只是思想保守，执行力也没他们那么强，最令人羡慕的是在移动互联网时代，他们才是网络的原住民，他们才是中国的未来和希望。  

  

因为年轻，因为有底气，所以他们敢于去做自己真心想做的事情，敢于犯错，积极的去解决问题，最重要的是肯为别人考虑。  

  

而且对于「赚钱」这件事，一点儿也不害羞，本来就应该大大方方的讲出来东西，我们这代老家伙却总是明面上称兄道弟，背地里比谁都算得精，最后两败俱伤 .....  

  

比如，今天突然收到了一则微信消息，看完有点泪奔啊。

  

想起当年自己好不容易加了个所谓的巨V大佬微信，结果半天憋不出来一句话，连个自我介绍发过去的勇气都没有，只配在朋友圈里点赞，就连给大佬朋友圈状态写个评论都要小心翼翼，生怕说错话被教育一番。

  

但，其实，好像这届95后根本没有这样的「自卑」和「纠结」，与生俱来的自信会传染给我这个老家伙。

今天夏夏写了一篇文章「 [ 房子炒过头了，挨锤了...
](http://mp.weixin.qq.com/s?__biz=MzAwNzM3NzU4OQ==&mid=2650707626&idx=1&sn=c23d7d40a6835efd999b21a630f0419e&chksm=83753170b402b8664835236705ad5bfdd49fb64e2a39d7613f956e673bb421a6c5164081f254&scene=21#wechat_redirect)
」，看完随手评论立马随手发了朋友圈，因为从文章里看到了很多来深圳的那些95后，他们只会选择自己真的喜欢的地方，碰到房产的问题不是去网络上追求公平和保护，而去找点业内人，耐心的请教别人，甚至去星球里知识付费学习相关的知识。  

  

解决问题，而不只是提一个问题和评价社会问题....  

  

昨晚深吹兄弟的直播「 [ 深圳的高房价真的能赶走人才吗？
](http://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649816361&idx=1&sn=676528687c1758cc843efa537912a607&chksm=8896726ebfe1fb78533b93dd6dd04e6da453ca01cf458280b6da3d9864311cde9d78156d7a41&scene=21#wechat_redirect)
」里也有两个90后，我就搞不懂差不多都从农村出来的， 人家怎么就那么大的勇气和自信留在深圳呢？  

  

看看昨天直播说话最少的新东方雅思老师 @建伟 的真实故事「 [ 而立之年，给人生一个机会
](http://mp.weixin.qq.com/s?__biz=MzIyNTk2MDQ2MA==&mid=2247484259&idx=1&sn=ffc9ba1e853c826b377af1a5f49533a0&chksm=e876f400df017d1603b36f7eccfadc4f92cb7822a19f5310f273d3b44a591800041d0713dff7&scene=21#wechat_redirect)
」，真心搞不懂那些家庭条件比建伟和Alex更好的90后们，你们在怕什么？....  

  

生在全球最好的国家，还有机会可以在全中国最好的城市享受自由和奋斗的权利，有啥可怕的呢？  

  

就算你拖家带口有了家室，有了孩子，还有老家的父母，深吹夫妇的故事「 [ 他带30岁的她来到深圳，一年后，她包养了他！
](http://mp.weixin.qq.com/s?__biz=MzI0ODgyNzQ4NQ==&mid=2247484047&idx=1&sn=cb3b7982765d88c8356f42629eca0eca&chksm=e99b99bddeec10aba038e9390c0d2cd4455aa138e07c747b0f7ddbc999f8f235643f71ee9548&scene=21#wechat_redirect)
」里写的那样，我们不是也把结果做出来了吗？  

  

好吧，再来个广告，当年听到Wendy博士的课程，才体会到原来不止我们，孩子的人生也需要规划，来扫码看看训练营....
