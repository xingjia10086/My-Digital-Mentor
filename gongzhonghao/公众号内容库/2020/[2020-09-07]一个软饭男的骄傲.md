#  一个软饭男的骄傲....

原创  星佳  [ 星佳是个小人物 ](javascript:void\(0\);)

__ _ _ _ _

还记得去年感恩节「 [ 感恩夜，许愿成为一个软饭男
](http://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649815992&idx=1&sn=25147dcfd22198c1c9c7044623558a0b&chksm=889674ffbfe1fde97e5f14e96455d0212514ffe2cbfc5714e945d17deb736ac03336b7b6d27d&scene=21#wechat_redirect)
」和今年的愚人节「 [ to 爱米：请照顾好我这个软饭男啊！
](http://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649816288&idx=1&sn=86bb2b4ad9b7c3d10a87de7b6e2330d6&chksm=889673a7bfe1fab131aad9b424d477b0b098fb74bd88921d540d041533e293a4f287c33f5f43&scene=21#wechat_redirect)
」 分别针对软饭男的主题写了两篇文章，公开了即将迈入30岁的星佳许下的大愿望，安心的成为一个软饭硬吃的全职爸爸。  

  

![](https://mmbiz.qpic.cn/mmbiz_jpg/tOFSAibJ5wqsfQZUGe6okyknH6qIgTpfeb2mNsBrumnmeFCPT6gjhEqiaAQZIs3kKCRc5GroegNh3XMibP2laW0fw/640?wx_fmt=jpeg)

  

其实起初看到这副漫画还有点不理解，为何，明明漫画里的男主角占了便宜还一副嫌弃的样子，可能这就是不自信的表现吧，而旁边身材不错的美女明明条件更优势却依然要拉一脸不屑却又不乐意松手小白脸一把，这就是助人助己的精神啊。

  

如果人人都献出一份爱，世界将变成美好的人间。

  

翻了下百科里的定义：  

  

> 有些  好逸恶劳
> ，不思进取的男人，希望以色相勾搭富有的女人并讨好她们，看女人脸色过活，从而让女人提供自己满意的的物质生活；还有些男人完全依靠妻子的裙带关系谋取利益。这样的男人行为俗称“
> 吃软饭  ”，他们又被称为“  小白脸  ”和“软饭男”。  
>  有些“能人”在蛰伏阶段，也靠女人提供衣食，他们也自嘲为软饭男。  百度百科

  

随着文明社会的发展，越来越多的词语开始有了多重含义，今天就瞎聊聊为何我这么安心的想做个软饭男？  

  

>  
>
>
> 一等：互助互利型
>
>  
>
>
> 这种组合多数是已婚夫妻档，男人有些头脑却时运不济，像李安。不是以占有女人财产为目的，想借助对方的人际关系和社会基础等优势，共谋福利共享收获。
>
>  
>
>
> 二等：以逸待劳型
>
>  
>
>
>
> 这一类人多数有情趣或气质，却没有特长或信心，喜欢小富即安的生活。一旦找到一个有稳定收入的女人，骨子里那点原始的积极性也会在女人的宽容和付出里消磨得荡然无存，心安理得地做起一个成功女人背后的男人。
>
>  
>
>
> 三等：薄情厚物型
>
>  
>
>
> 这一型软饭男就是把自己当女人来活，软得最彻底最直接，软到骨头里。
>
>  
>

  

你看，即使是「软饭男」这么一种如此朴素的愿望，居然也有三等层次，而争强好胜的星佳肯定要勇争第一，吃软饭也要成为最杰出的那位。

  

之前在「 [ 如何慢慢变富，请看深吹夫妇....
](http://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649816083&idx=1&sn=d12853ffec8539598439f83fcd9cec9a&chksm=88967354bfe1fa4257b0add4c41fdc0a4323d558266f821026f90761dbadc1e0617e7c121cce&scene=21#wechat_redirect)
」详细写了到深圳这两年多到底发生了什么，一直相信战略的勤奋叠加战术的执行才是完美的计划，所以，就算要吃软饭，也要携手跟另一半把饭做的精致，做出不仅果腹而且秀色可餐的模样。  

  

在「 [ 深吹夫妇: 人生中最重要的三个决定
](http://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649816179&idx=1&sn=2cacb847ec540217823f74b3197ffe92&chksm=88967334bfe1fa22512acfc5bccd9dc51087889681d67df8f7a98dd0a05bb04cb02ee70940fd&scene=21#wechat_redirect)
」里，铺好的框架都是软饭男的思考，而定下大纲之后，执行部分的成果就脱不开爱米的能干。  

  

自从定居到深圳后，一个人打工上班，另一个人创业突破天花板，把家庭活成一家有愿景的公司，而两个人也需要各自分工，就像一支厉害的球队，一定是场下睿智的教练跟场上锋芒毕露的球员，一起配合，才能给观众奉献一场精彩的比赛。  

  

而随着暂停了一个月的思考「 [ 按下人生的暂停键，给自己一个拼命的理由
](http://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649816735&idx=1&sn=b56d87bdc327b61fc2fee1bfcc2dba5d&chksm=889671d8bfe1f8ce524cb806d5a939c620657f7930ecaf9061524bd83ca98eea7230386bfd92&scene=21#wechat_redirect)
」后，我们开始明白一些朴素的道理，比如哪里文明就去哪里，哪里尊重市场就去哪里，哪里可以发挥个人的才华就去哪里，哪个赛道可以更容易的帮我们达成目标就去哪里。  

  

看着置身杭州，进入下一个人生阶段的爱米在朋友圈里自信的晒着这座人口流入超过深圳的城市，软饭男也按捺不住内心的期待，或许也应该做点什么了。

  

![](https://mmbiz.qpic.cn/mmbiz_jpg/tOFSAibJ5wqsfQZUGe6okyknH6qIgTpfe18uafkh5xtRfWvYOzTQ1Kg2bZ4ezWKmTprsu3kjVEe3MnfLl1k26WQ/640?wx_fmt=jpeg)

  

突然想到「延迟满足」这个词，
就跟今天跟大家分享「软饭男」这个词的观点一样，即使像很多大佬所谓的致富秘诀，可能在某个时间点也并不适合自己的情况，星佳一直相信个体所能做出来的当下的选择，一定是最符合当下的个体利益的。  

  

于是，接下来等爱米从杭州出差回来后，要开始花点时间去了解杭州这座城市，去搞明白为何这么多人口流入，去明白浙商的历史，去搞清楚为何杭州会诞生那么多厉害的公司？

  

然后，重仓这个城市，复活当年深吹时候的激情，开始享受城市发展带给我们这一代的发展红利，顺势而为。  

  

相信爱米的判断，扮演软饭男的身份，做好思考的思考，省下的交给时间会给我们答案。
