#  用了这么多年的Mac，后悔没有早点知道这些干货！

原创  星佳  [ 星佳是个小人物 ](javascript:void\(0\);)

__ _ _ _ _

随着 Apple 发布了一系列的新品，叠加教育优惠和24期免息分期，曾经被认为要割肾才能买得起的苹果全家桶如今确实已经成了很多职场人和学生党的标配，毕竟
macOS 上大量优秀的软件是真的可以提升我们的生产力。

  

![](https://mmbiz.qpic.cn/mmbiz_png/tOFSAibJ5wqtatgKznYEVxP1Xj2t5B3YgDxu1h5LekOZbr7uqlfjz5q1CUV4Y6x3IeAstBMDKRgVRZvnfFs8GnQ/640?wx_fmt=png)

  

去年买 MacBook Air 写了一篇「 [ 一年一次机会，教你最便宜的在官网买苹果全家桶
](http://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649815561&idx=1&sn=b7f7111f0e3b4c636c492fa2348a7428&chksm=8896754ebfe1fc5889a52441565f7c3b32bb3df1aafe43bbccbbfc801d1d9d5729c2d50022f6&scene=21#wechat_redirect)
」，结合信用卡的24期无息分期和各大电商的618活动折扣，买到苹果产品真的并没有那么贵。  

  

如果你还是喜欢用 Windows ，星佳依然坚持认为你可以试试 macOS ，因为那是另一种思维方式。  

  

那么，硬件有了，作为 macOS 用户如何找到优质的软件呢？  

  

一般有了 Apple ID 之后都会去 App Store 下载软件，但是因为众所周知的原因，某些特殊功能的 App
是不允许在官方市场上架的，比如星佳最喜欢的磁盘空间清理软件 - 腾讯柠檬清理 来举例子：  

  

![](https://mmbiz.qpic.cn/mmbiz_png/tOFSAibJ5wqtatgKznYEVxP1Xj2t5B3YgD1s8KJvQdDX8FBygu270OBcpJjY1zYdkqfP2fyLvXp63WpgCgxfDcQ/640?wx_fmt=png)

  

而很多新手同学可能就有疑问了，如果不去官方市场买，岂不是有可能中招流氓软件了？而且 Mac 也都不装杀毒软件，岂不是很危险？

  

但其实在 系统偏好设置 里的 安全性与隐私 里有个安全性设置，大家为了安装完整版的 .dmg 格式的应用，勾选如图的选项即可。

  

![](https://mmbiz.qpic.cn/mmbiz_png/tOFSAibJ5wqtatgKznYEVxP1Xj2t5B3YgRDVAP6B21RIwIKHGDYoRLWZ9sknFTfwHtEu9LgDlyh44IlPSYJ7TbQ/640?wx_fmt=png)

  

腾讯柠檬清理应用有 应用卸载，大文件清理，重复文件清理，相似照片清理等功能，支持开机自启动，和状态栏的实时显示。

  

![](https://mmbiz.qpic.cn/mmbiz_png/tOFSAibJ5wqtatgKznYEVxP1Xj2t5B3Ygfq3PlFIJNaK7IYv2ToCq1SE1ZwofOnMTVVCdVOKnod5SnicFiauQVj1A/640?wx_fmt=png)

  

可以在顶部状态栏清晰的释放内容，和查看CPU温度，磁盘占用，和网络速率等等.....

![](https://mmbiz.qpic.cn/mmbiz_png/tOFSAibJ5wqtatgKznYEVxP1Xj2t5B3YgdEqLfe3usVWZqRlDOLd67uulLdQ1sPEKgvNP1STRH6TRjxnibDM4U8A/640?wx_fmt=png)
