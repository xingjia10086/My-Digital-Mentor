#  玩世界顶级AI工具的学习笔记

原创  星佳  [ 星佳是个小人物 ](javascript:void\(0\);)

__ _ _ _ _

这篇文章算个学习笔记，总结下最近入门AI编程和使用AI工具的一些心得吧。
第一个很大的启发，别看新闻稿和自媒体人怎么说，先花时间去真的用一用那些世界顶级的工具，花了那么多钱，聚集那么多聪明人做出来的工具，我们不去用用真是亏大了啊。
就算有一些门槛，但在星佳看来，那些成本时间和金钱的投入绝对是划算的。  于是今天由衷的感叹到过去自己的愚蠢，发了条朋友圈。
![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqs9PGl4RDJ2L9EbVgwTWfk4zKsD83xhjyOdudSegMDc9ODUyS7rBL7MMooTwpue13yWbDAP2xks9w/640?wx_fmt=png&from=appmsg)
命令行工具编程和图形化工具编程，没有谁优谁劣，就跟星佳一直强调的观点，iPhone也好，安卓也好，Windows也好，macOS也好，都用，都去体验，然后取长补短才是使用工具的人应该有的执行力。
![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqs9PGl4RDJ2L9EbVgwTWfk42OBXxQibFOJ7hSxe5CT3kRMlA9xIyNBo9S2VMrcZxIF9POUuXeXSVNQ/640?wx_fmt=png&from=appmsg)
现在电脑上在浏览器端同时使用ChatGPT，Claude，Gemini和Grok等。
传有限的文件，或者命令行的报错截图，长视频和代替部分搜索引擎的能力，但终究传给他们的文件大小和数量基本都会被被限制的死死的，所以本地部署的API方式更适合处理多文件的情况，尤其辅助星佳这类文字内容工作者，指望浏览器和客户端连实习生的能力都赶不上。
于是，必须使用命令行的方式，在IDE开发环境里配置好大模型的API才能处理上千个文件，或者几百万字星佳写的原创内容。  Cursor的IDE配合CLI 的
Gemini 和 Cladude Code 解决了直接和浏览器对话，无法上传所有文件的问题，本地电脑部署工程，调用API互动的方式真的很高效啊！
理论上，做企业内部服务，也只能靠本地部署来解决问题，所以不如刚开始麻烦点，本地环境搭建完毕，剩下的无非学学基本的命令行操作，然后开始天马行空的创作吧。
同样一个问题，丢给不同的AI给你参考答案，甚至会有意外惊喜，这就像你有全球最牛的智囊团给你参考意见，当然最后做决定的还是你自己！
然后经过几天的深度使用，给感兴趣的同学布置几个任务：
1.先成功的注册好这四大AI工具，在浏览器端问他们同样一个你的行业内的专业问题，对比下他们四家给出答案的质量，毕竟你是内行人。
2.一定要学会部署命令行编程，说个秘诀，只要你安装好
Cursor或者Trae等AI赋能的IDE编程环境，直接把官方安装文档的链接和指令丢给IDE，一切顺利的情况下，你需要点几次权限的同意回车键，他们就会帮你搞定各种让新手糟心的环境配置。
直接在Chat模式下，用自然语言说出你的需求，授予IDE一些基本权限后即可交给他去操作下载，安装和配置等等的基础工作。
3.学会在你自己新建的测试工程里，打开 Gemini 和 Claude 的命令行模式，在黑糊糊的代码库里，用一些基础命令去操作文件，发号施令！
4.任何报错和不懂的，直接截图给浏览器端的几个AI工具，他们会给你解决方案的。
5.任何看到的文章，尤其是新闻，你都可以发给这几个AI工具，帮你进行事实核查和逻辑漏洞检测，有些新闻编的和小说一样，星佳个人觉得也是每个人必须会的一种工具使用方式，去判断垃圾信息和谣言内容的能力！  

>
> 一直有个需求总想着满足，因为这些年写的公众号太多了，基本没有本地备份的习惯，每次想搜索自己的一些历史文章的观点时只能用微信自带的公众号搜索功能，但体验真的不好。
>
>  
>
>
>
> 然后元宝出来后，又出了一个元器功能，原理就是授权给这个智能体之后，可以在公众号后台和读者互动，其实这就是星佳一直想要的功能，但自己亲自用了下之后发现，基本没啥用，连我作为作者想查询自己历史文章的某些信息都搞不定。
>
>  
>
>
>
> 于是随着GPT5.0发布后，就有了个想法，把历史文章都下载下来，大概有2000多篇，每一篇文章都是MD,HTML、PDF格式，合并成一个文件，之后传给Gemini
> 和 ChatGPT 看看谁的长记忆功能更能解决我的问题。
>
> 星佳&实习GPT，公众号：星佳是个小人物 [ 公众号搜索不好用，那就自己做一个吧！

一直有个心愿，把写了这么多年的文章分别投喂给这几个世界顶级的大模型，然后基于他们对星佳的了解继续互动，甚至给建议，传统的浏览器和客户端模式，根本无法上传几千个文档和数百万字的内容（也许是我不会，请指教）
直到开始使用命令行的方式，大模型得到我的授权后，直接在我的电脑文件夹本地操作文件，增删改查，做交互，这种才是agent啊！
这两天一直在研究这件事，也真的觉得AI的强大和可怕，根据他们对星佳提供的几百万字的思考后做出的回答，以后很难离开他们了，很多事第一时间必须是寻求他们的帮助，相信做过几个月，半年，随着自己使用工具的深入，应该会有更大的感叹吧，到时候再和大家分享呗。
![](https://mmbiz.qpic.cn/sz_mmbiz_png/tOFSAibJ5wqs9PGl4RDJ2L9EbVgwTWfk4xODtJjCbzbP7UPw6XMGHBdwoHgiaJ5o3KIhUvNInJ3KETMPFibbrRU7g/640?wx_fmt=png&from=appmsg)
今天又开始玩马斯克的
Grok，专家模式下居然对我提的问题出来两个参考答案，甚至某些问答直接输出表格为了让我的看的更直观，据说特斯拉餐厅也开幕了，机器人每天给排队的顾客做爆米花吃，期待特斯拉的股价再次让华尔街震惊吧。
对了，最近ETH疯涨，切忌盲目投资啊，但一定要投点钱躬身入局，起码走一遍了流程去自己体验，就跟星佳居然开始使用命令行的方式和AI交互了，看多少新闻稿都不如自己动手去玩玩。
