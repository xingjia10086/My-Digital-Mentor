早晨又复习了几个概念

  

我们所谓的语言能力，基本上 99% 都是复述能力。

  

a ) 复述，就是用你自己的话、对自己再讲一遍，不仅如此，还要讲很多遍，并且还要换很多种说法、换很多个角度，还要时不时就给自己再“复述”一遍。

  

这里的要点 至少有4个：

  

一是用自己的话；二是要讲很多遍；三是要多角度；四是要 时不时

  

注意到我们的注意广度，用爱时间 App 记录时间花销，联系到了作者解决了 iphone 屏幕小部件没显示的问题，很简单重启手机即可

  

  

b ) 所谓的 “元认知”，就是 “关于认知的认知”，“关于思考的思考”

  

susan 老师推荐了电子书《学习之道》《Beyond FeeLings》和《Build》

  

《学习之道》作者下棋过程中调用元认知的小故事，思考的注意力被对方敲棋的动作和声音干扰

  

又找到了一个 Github 仓库

还有豆瓣里李笑来推荐的书单和影单

  

等研习社结束要好好啃啃，脑袋里需要投喂优质的内容，哪怕是娱乐也要看好的电影和小说

  

  

“元认知”的核心就是 “判断依据” 我们要凭借这些依据去做质量判断。

  

比如，刚才最开始切入的，自己的注意广度是否符合“真学习”的标准了，这就是一个“判断依据”

  

元认知 落地在纸面，就是自检清单，或者叫 检查清单 ，检查列表。

  

为自己的自学目标制定一个 “检查列表”，罗列学好做好的关键衡量指标，这里的“自检清单”、或者
叫“检查列表”，在1000小时英语学习上，就是笑来老师给出的每日任务，包括语音塑造、记忆扩展上的各种细节要求。

  

这个列表就是你调用并集中注意力的目标。

  

之前看 笑来老师给出的 1000小时 英语发音教程，只是教程，但现在你不一样了，你知道了那个其实是你的自检清单，是调用元认知要去搭建的新网络
新链接的标准，你用 ENjoy的学习过程 ，就是 不断去自检的过程 得练习到成为你的肌肉记忆。

  

  

在《教练的真相》中，笑来老师专门提及了“把事做好”的2个标准，

  

● 自检列表足够细致完善（不断补充）

  

● 衡量标准足够严格（习惯于略微再抬高一点标准）

  

终其一生，所谓的 “进步”，也都可以归结为 “自检列表的进一步完善”
，一个是从数量的多少上、从角度的多样上去说，另一个是从标准严格度上去说，简化来看，其实就是对“自检列表的进一步完善”

  

  

取乎其上，得乎其中。取乎其中，得乎其下。

  

要按最高标准严格要求自己。

  

冠军教练和普通教练的区别！

  

  

c ) 努力就是短时间内足量重复

  

短时间，足量重复是如何被量化的？

  

就是在“在一年内投入1000小时注意力到某个技能上”的时候，要致力于“新建连接、新建网络”，而不是“调用近似连接、近似网络”

  

升级版的《多语的真相》 https://1000h.org/

  

网站上的内容，通过读例句讲清楚何为 “新建连接、新建网络”，之前从来没有想过要通过 发音、连读、韵律和声调 去学习英语，于是也更理解了 Enjoy App
为何就是把事做厚，把事情做到极致就是优秀。

  

Serendipity

  

We gotta treat it like gold

  

所谓真学习，每天 3 小时以上，持续注意力集中

  

因为，绝大多数人在学习效果上的差异，几乎
99.99%，其实全都来自于这一个小小的细节，就是你能否调用自己的元认知，即：能否甄别出自己是否正在调用近似连接、近似网络？

  

只要能够甄别，你就能通过努力，即，短时间内足量重复，或者说，用一年1000小时的注意力投入、去新建连接、去新建网络！

  

不知道，就注意不到；但知道了注意到了，还需要练习

  

关于要建立新网络、创建新链接，其实不止是在 某个具体的技能上 ，而是体现在你想要“迭代”的任何方面，用 Enjoy
练习英语，其实就是练脑，健脑，不断建立新网络，建立新链接，持续迭代。

  

所以 必须投入注意力，必须足量练习 的核心意义，你是没有办法拿着旧地图找到新路径，你得让新地图成为你的肌肉记忆。

  

  

d ) 生产 - 找准生产的生态位

  

“时间”是我们的“终极生产资料”，我们应该珍惜时间，要把时间花到自学上。自学，是开发“时间”这个终极生产资料的最佳方式，甚至是唯一的方式

  

与此同时，自学这个动作，就是往你的时间管道里倾注、填充“注意力”

  

“财富来自于生产”中的“生产”，可以理解为广义上的生产；（ 制造，交易和发现 发现价值洼地 投资）

  

而“要通过生产、销售、投资去创造物质财富”中的“生产”，可以理解为对应着“制作”的狭义上的生产（生产，销售和投资）
