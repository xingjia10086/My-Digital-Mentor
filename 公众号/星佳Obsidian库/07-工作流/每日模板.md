# 每日工作流模板

> 基于星佳的写作习惯和高效模式设计
> 参考：四象限工作法、深度工作、家族优先原则

---

## 晨间例行（6:00-9:00）

### 起床与身体唤醒
- [ ] 起床，喝一杯温水
- [ ] 简单拉伸/运动（15分钟）
- [ ] 查看睡眠质量

### 晨间写作（黄金时间）
- [ ] **写作60-90分钟**（最重要）
  - 类型：观点文 / 故事文 / 碎碎念 / 家书
  - 参考：[[写作助手模板]]
  - 目标：完成初稿或大纲

### 家庭早餐
- [ ] 与小星星/小太阳互动
- [ ] 与爱米同步今日计划
- [ ] 检查家族群消息

---

## 上午深度工作（9:00-12:00）

### 9:00-10:30 第一深度时段
- [ ] **孟母三迁核心业务**
  - 客户咨询回复
  - 香港身份申请跟进
  - 团队协调

### 10:30-10:45 休息
- [ ] 离开屏幕，活动身体
- [ ] 补充水分

### 10:45-12:00 第二深度时段
- [ ] **内容创作或学习**
  - 文章修改完善
  - 视频号内容策划
  - 阅读/学习（AI、教育、房产）

---

## 午间（12:00-14:00）

### 午餐与家庭时间
- [ ] 家庭午餐（如在家）
- [ ] 陪小星星玩耍/学习
- [ ] 短暂午休（20-30分钟）

---

## 下午工作（14:00-18:00）

### 14:00-15:30 第三深度时段
- [ ] **对外连接**
  - 客户电话/视频会议
  - 合作伙伴沟通
  - 直播（如安排）

### 15:30-15:45 休息
- [ ] 茶歇
- [ ] 简单活动

### 15:45-17:30 第四深度时段
- [ ] **项目推进**
  - 长期项目（课程、文章系列）
  - 家族事务（太奶、晚辈沟通）
  - 投资决策研究

### 17:30-18:00 收尾
- [ ] 回复微信消息
- [ ] 整理今日工作
- [ ] 更新[[任务看板]]

---

## 晚间家庭时间（18:00-21:00）

### 晚餐
- [ ] 家庭晚餐（核心家庭时间）
- [ ] 与爱米深度交流（夫妻关系维护）

### 亲子时间
- [ ] 陪小星星阅读/游戏
- [ ] 小太阳照顾（如需要）
- [ ] 与太奶/父母通话

### 个人时间
- [ ] 阅读/电影（放松）
- [ ] 整理明日计划

---

## 晚间复盘（21:00-22:00）

### 情绪追踪
- [ ] 更新[[心理状态追踪表]]
  - 今日整体情绪（1-10分）
  - 主要情绪状态
  - 触发事件

### 写作复盘
- [ ] 今日写作字数：___
- [ ] 发布文章：___篇
- [ ] 读者互动：___条

### 明日规划
- [ ] 确定明日最重要的1-3件事
- [ ] 检查日程安排
- [ ] 准备明日写作主题

---

## 周末特别安排

### 周六
- [ ] 家庭活动日（外出/旅行）
- [ ] 房产考察（如需要）
- [ ] 直播/视频号录制

### 周日
- [ ] 周度回顾（见[[每周回顾模板]]）
- [ ] 下周规划
- [ ] 家族沟通（宁夏家人）

---

## 紧急情况处理

### 如爱米需要支持
- 优先调整日程支持爱米
- 夫妻共同事务优先

### 如孩子有事
- 家庭优先原则
- 工作可调整

### 如客户紧急需求
- 评估紧急程度
- 与爱米协调

---

## 使用说明

1. **打印此模板**或保存为每日笔记模板
2. **晨间勾选**：每天从晨间例行开始
3. **灵活调整**：根据当日实际情况调整
4. **周末复盘**：回顾本周执行情况
