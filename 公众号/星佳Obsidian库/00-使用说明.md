## 🎯 快速开始

### 第一步：打开主页
点击 `主页.md` 开始导航

### 第二步：选择写作模板
进入 `02-素材库/模板/` 选择适合的模板：
- [[观点文模板]] — 表达观点
- [[故事文模板]] — 讲述故事
- [[碎碎念模板]] — 日常记录
- [[家书模板]] — 给孩子的话

### 第三步：复制模板到01-原创文章
1. 复制模板内容
2. 在 `01-原创文章/` 新建文件
3. 命名格式：`[日期]文章标题.md`

### 第四步：写作
- 参考 `02-素材库/金句库/核心金句.md` 插入金句
- 使用 `[[文章标题]]` 格式引用历史文章
- 文末添加"引用文章列表"

### 第五步：发布到公众号
- 完成的文章直接复制到公众号编辑器
- 格式已优化为公众号样式

## 🔗 双链系统说明

### 什么是双链？
Obsidian的双向链接功能，用 `[[文章名]]` 格式创建可点击的链接

### 如何使用？
1. 在文章中输入 `[[`
2. Obsidian会自动提示已有文章
3. 选择要引用的文章
4. 点击链接可直接跳转

### 示例
```markdown
正如《 [[我的家学6个早点和6个国际化]] 》里写的...

// 文末列表
## 引用文章列表
- [[夫妻关系才是人生逆袭的加速器]] — 本文核心观点来源
- [[感谢我的儿子是个学渣]] — 教育理念支撑
```

## 🏷️ 标签系统

### 如何使用标签？
在文章末尾添加：
```markdown
#待发布 #观点文 #教育规划
```

### 常用标签
- **状态标签**：#待发布 #已发布 #草稿
- **类型标签**：#观点文 #客户故事 #碎碎念 #家书
- **主题标签**：#教育规划 #香港身份 #深圳买房 #夫妻关系
- **核心标签**：#六个早点 #六个国际化 #孟母三迁
- **人物标签**：#小星星 #小太阳 #软饭男

## 📝 写作规范

### 文章编号
每篇文章底部标注：
```
这是星佳和爱米的第 XXXX 篇原创
```

### 固定结尾
所有文章必须以以下文字结尾：
```
记录生活，做出结果，真诚分享，就一定可以帮有缘的读者抄作业少走人生的弯路：
```

### "临在时刻"板块
每篇文章结尾必须有"临在时刻"板块：
```markdown
临在时刻

—

（总结观点）

nn

（号召行动或留下问题）

nn

（引用太爷遗言或金句）
```

进入 `02-素材库/金句库/核心金句.md`

## 💡 写作提示

### 如何保持风格一致？
参考 `03-AI协作记录/写作风格规范.md`

### 需要引用某篇文章但找不到索引？
1. 搜索原文路径 `../202X/[文件名].md`
2. 在 `04-原文引用/` 创建索引文件
3. 使用双链引用

### 如何记录新的协作任务？
在 `03-AI协作记录/` 创建新文件：
```
任务记录-XXX-任务名称.md
```

## ⚡ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 快速打开文件 |

| `Ctrl+E` | 切换编辑/预览模式 |
| `[[` | 创建双链 |
| `#` | 创建标签 |
