### 离开腾讯的决策逻辑

**触发事件**：
> 爱米称星佳为"废物老公"（2021.7.2）
> - 虽然收入不错但无法深度参与家庭
> - 时间被工作占满，无法陪伴

**决策框架**（当时）：
```
1. 经济安全：副业收入已可覆盖基本生活
2. 时间自由：可以深度参与育儿
3. 家族使命：孟母三迁可以服务更多人
4. 个人价值：写作和咨询更有成就感
```

**2021年成果**：
- 收入超过腾讯时期
- 时间灵活性大幅提升
- 小太阳出生，深度参与育儿

### 业务升级

```
2020年：课程+咨询
2021年：+ 香港身份规划（萌芽）
```

### 学渣论（2024年1月22日）⭐⭐⭐

来自《感谢我的儿子是个学渣》：

**核心观点**：
> "学渣只是不适合某些体制，偏科才是大部分学渣的真实状况，但是并不妨碍他们有擅长的技能"

> "多去看看名人传记吧，学习好坏根本不重要"

**与传统观念的差异**：

| 传统观念 | 学渣论 |
|----------|--------|
| 学习好=优秀 | 学习好≠未来成功 |
| 补差科 | 扬长项 |
| 挤高考 | 多路径规划 |
| 焦虑驱动 | 接纳+规划 |

**对孟母三迁的意义**：
- 从"帮孩子成功" → "接纳孩子+规划路径"
- 更符合现代教育理念
- 区分"孟母三迁"与"鸡娃"
