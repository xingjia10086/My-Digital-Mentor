# 孟母三迁客户旅程地图

> 从认知到长期价值的完整服务流程
> 基于2017-2025年服务3000+家庭的经验提炼

---

## 旅程总览

```
认知阶段 → 考虑阶段 → 决策阶段 → 服务阶段 → 获批阶段 → 长期阶段
(1-3月)    (1-2月)    (1-2周)   (3-12月)    (1-2月)    (持续)
```

---

## 第四阶段：服务阶段（核心）

### 时间跨度
- 3-12个月（申请到获批）

### 客户触点

| 触点 | 形式 | 星佳动作 |
|------|------|----------|
| 资料收集 | 线上/线下 | 协助整理材料 |
| 文书撰写 | 专业文案 | 赴港计划书等 |
| 递交申请 | 官方渠道 | 确保无误 |
| 进度跟进 | 定期更新 | 每周状态同步 |
| 补件通知 | 紧急响应 | 快速处理 |
| 面试辅导 | 模拟面试 | 专业培训 |

### 详细服务流程

#### 优才申请流程
```
1. 评估打分（1周）
   - 基础条件评估
   - 分数预估
   - 成功率分析

2. 资料准备（2-4周）
   - 学历证明
   - 工作证明
   - 推荐信
   - 赴港计划书

3. 递交申请（1周）
   - 在线填写
   - 材料上传
   - 确认提交

4. 等待审批（3-9个月）
   - 定期跟进
   - 补件处理（如需要）
   - 面试辅导（如需要）

5. 获批通知（1周）
   - 收到原则上批准
   - 办理签证
   - 预约身份证
```

#### 高才申请流程
```
1. 资格确认（1周）
   - A/B/C类确认
   - 材料清单

2. 资料准备（1-2周）
   - 简化材料

3. 递交申请（1周）
   - 在线申请

4. 等待审批（2-4周）
   - 快速通道

5. 获批（1周）
   - 办理手续
```

### 客户行为
- 准备个人资料
- 配合文书撰写
- 参加辅导培训
- 等待审批结果

### 客户心理曲线
```
签约后：期待 ↑
资料准备：焦虑 →
递交后：忐忑 →
等待期：焦虑 ↓（随时间延长）
补件通知：紧张 ↑
获批前：极度焦虑 ↑↑
获批后：兴奋 ↑↑↑
```

### 星佳交付物
- 专业文书服务
- 定期进度更新
- 24小时紧急响应
- 心理辅导支持
- 补件快速处理
- 面试专业辅导

### 关键里程碑
- [ ] 资料收集完成

## 第五阶段：获批阶段

### 时间跨度
- 1-2个月（获批到落地）

### 客户触点

| 触点 | 形式 | 星佳动作 |
|------|------|----------|
| 获批通知 | 邮件/电话 | 第一时间告知 |
| 签证办理 | 指引 | 详细步骤说明 |
| 香港开户 | 陪同/推荐 | 银行资源对接 |
| 身份证预约 | 协助 | 抢号/预约 |
| 香港租房 | 资源推荐 | 房产中介对接 |
| 子女入学 | 学校推荐 | 教育资源对接 |

### 详细流程
```
1. 收到原则上批准（1周）
   - 确认获批类别
   - 说明后续步骤

2. 办理逗留签证（2-4周）
   - 内地手续指导
   - 香港签证办理

3. 首次入境（1周）
   - 入境激活
   - 办理身份证预约

4. 领取身份证（1-2周）
   - 陪同办理（可选）
   - 领取证件

5. 后续服务（持续）
   - 银行开户
   - 子女入学
   - 租房买房
```

### 客户行为
- 办理内地手续
- 安排香港行程
- 准备子女转学
- 规划香港生活

### 客户心理
- **兴奋**：终于获批了！
- **忙碌**：好多事情要处理
- **期待**：新生活即将开始

### 星佳交付物
- 获批后指引手册
- 香港生活指南
- 资源对接清单
- 陪同服务（可选）
- 后续咨询支持
