# 快速决策检查清单

> 基于6个早点和6个国际化的日常决策工具
> 10秒检查 → 1分钟框架 → 完整流程

---

## 第一层：10秒决策检查

### 快速三问

面对任何决策，先问自己：

**问1：这符合6个早点吗？**
- [ ] 有利于早点结婚/维护婚姻？
- [ ] 有利于早点生娃/养育？
- [ ] 有利于早点买房/资产配置？
- [ ] 有利于早点来一线城市/发展？
- [ ] 有利于早点输出/积累影响力？
- [ ] 有利于早点配置保险/风险管理？

**问2：这有利于6个国际化吗？**
- [ ] 身份自由？
- [ ] 资产配置？
- [ ] 子女教育？
- [ ] 医疗保障？
- [ ] 收入来源？
- [ ] 社交网络？

**问3：最坏结果能承受吗？**
- [ ] 财务损失在储备金范围内？
- [ ] 不影响家庭核心利益？
- [ ] 不损害身体健康？
- [ ] 不破坏关键关系？

### 10秒决策结果

如果3个都是✅ → **推进**

如果有1-2个⚠️ → **进入1分钟框架**

如果有❌ → **暂停或放弃**

## 第二层：1分钟决策框架

### 适用场景
- 中等重要性决策
- 时间压力不大
- 需要快速评估

### 决策矩阵

| 维度 | 权重 | 选项A | 选项B | 选项C |
|------|------|-------|-------|-------|
| 符合6个早点 | 20% | /10 | /10 | /10 |
| 符合6个国际化 | 20% | /10 | /10 | /10 |
| 财务回报 | 15% | /10 | /10 | /10 |
| 时间成本 | 15% | /10 | /10 | /10 |
| 风险程度 | 15% | /10 | /10 | /10 |
| 机会成本 | 15% | /10 | /10 | /10 |
| **加权总分** | 100% | | | |

### 快速计算

```
选项A总分 = (早点×0.2) + (国际化×0.2) + (财务×0.15) +
           (时间×0.15) + (风险×0.15) + (机会×0.15)

最高分者 → 优先选择
```

### 1分钟检查清单

**准备阶段**（15秒）
- [ ] 明确决策问题
- [ ] 列出2-3个选项

**评估阶段**（30秒）
- [ ] 快速打分（直觉）
- [ ] 计算加权分

**决策阶段**（15秒）
- [ ] 选择最高分
- [ ] 确定下一步行动

## 第四层：重大决策完整流程

### 触发条件
- 涉及金额 > 100万
- 影响时间 > 1年
- 涉及家庭成员 > 2人
- 不可逆决策

### 使用工具
[[决策框架工具]]（完整版）

### 流程
```
1. 问题定义（1天）
   - 明确决策问题
   - 设定决策期限

2. 信息收集（3-7天）
   - 市场调研
   - 专家咨询
   - 案例研究

3. 方案制定（2-3天）
   - 2-3个可行方案
   - 每个方案详细分析

4. 评估决策（1-2天）
   - 使用完整决策框架
   - 与爱米深度沟通
   - 家族会议（如需要）

5. 执行计划（1天）
   - 详细执行步骤
   - 时间节点
   - 风险预案

6. 复盘机制
   - 3个月小复盘
   - 6个月中复盘
   - 1年大复盘
```

## 附录：决策常见陷阱

### 1. 沉没成本陷阱
**表现**："已经投入了这么多，不能放弃"
**破解**：只考虑未来收益，不考虑已投入成本

### 2. 从众心理
**表现**："大家都在做，应该没错"
**破解**：是否符合自己的6个早点/国际化

### 3. 过度自信
**表现**："我一定能成功"
**破解**：最坏结果能否承受

### 4. 信息过载
**表现**："需要更多信息才能决定"
**破解**：设定决策 deadline，够用即可

### 5. 情绪化决策
**表现**："当下冲动决定"
**破解**：使用检查清单，延迟重大决策24小时
