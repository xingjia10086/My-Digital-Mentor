# 写作助手模板

> 基于星佳2017-2025年1127篇文章的写作风格提炼
> 四种核心文章类型：观点文、故事文、碎碎念、家书

---

## 一、观点文模板

### 适用场景
- 表达对一个话题的深入思考
- 分享方法论或认知框架
- 回应热点事件（但要有独特视角）

### 结构框架

```
1. 开篇钩子（1-2段）
   - 个人经历切入 或 反常识观点
   - 例："很多人问我为什么离开腾讯，其实我想问的是..."

2. 问题/背景阐述（2-3段）
   - 交代这个议题为什么重要
   - 常见误区或大众认知

3. 核心观点展开（3-5段）
   - 分点论述，每段一个子观点
   - 结合个人案例或数据支撑
   - 金句穿插

4. 方法论/行动建议（2-3段）
   - 不是纯鸡汤，要有可操作性
   - 结合自身实践经验

5. 结尾升华（1段）
   - 回到个人，引发共鸣
   - 或抛出开放性问题
```

### 星佳风格特征
- **开篇方式**：用具体场景或对话切入，避免宏大叙事
- **论证风格**：个人经历+数据/案例+认知提炼
- **金句密度**：每篇3-5个值得被转发的金句
- **结尾风格**：回到自身，示弱或示真，引发共鸣

### 参考文章
- 《早点结婚，然后来大城市奋斗吧》（2019）
- 《感谢我的儿子是个学渣》（2024）
- 《趁着35岁前抓紧去冷冻胚胎》（2025）

## 二、故事文模板

### 适用场景
- 记录重要人生事件
- 客户案例或家族成员故事
- 阶段性总结或回顾

### 结构框架

```
1. 时间锚点（1段）
   - 明确的时间点，制造代入感
   - 例："2023年2月20日凌晨3点，我接到父亲的电话..."

2. 人物/背景介绍（1-2段）
   - 关键人物的基本信息
   - 与星佳的关系定位

3. 事件经过（按时间或逻辑展开）
   - 有起承转合
   - 细节描写（对话、场景、心理活动）
   - 冲突点或转折点

4. 思考/启示（2-3段）
   - 从这个故事中提炼的认知
   - 与读者生活的连接点

5. 后续/展望（1段）
   - 故事的延续或行动方向
```

### 星佳风格特征
- **叙事视角**：第一人称亲历者视角
- **情感节制**：不过度煽情，用事实打动人
- **细节选择**：选有代表性的场景和对话
- **标签体系**：文末加话题标签

### 参考文章
- 《1941.10.7-2023.2.20 爷爷，我想你了》（2023）
- 《带父辈带晚辈带整个家族来深圳》（2023）
- 《瑞智：从宁夏到深圳的四代人迁徙》（案例文）

## 三、碎碎念模板

### 适用场景
- 日常思考片段
- 不用太完整的观点
- 与读者建立日常连接

### 结构特征

```
- 篇幅较短（500-1000字）
- 不追求完整结构
- 可以是一个观察、一段对话、一个顿悟
- 语气更随意，像朋友聊天
- 常有自我调侃
```

### 星佳风格特征
- **话题来源**：日常生活观察、带娃经历、工作细节
- **语气特点**：自嘲、真实、不装
- **价值点**：即使碎碎念也要有"信息增量"
- **互动感**：常直接对读者说话

### 参考文章
- 《今天被爱米说了...》系列
- 《小星星的学渣日常》系列
- 《孟母三迁的日常》系列

## 四、家书模板

### 适用场景
- 写给特定家人（小星星、爱米、家族晚辈）
- 记录家族历史和家学
- 重大人生节点的嘱托

### 结构框架

```
1. 称呼（个性化）
   - "小星星：" / "爱米：" / "致家族晚辈："

2. 写作背景（1段）
   - 为什么在这个时间点写这封信
   - 触发事件

3. 回忆/叙述（2-4段）
   - 家族历史
   - 个人经历
   - 相关故事

4. 核心嘱托/期望（2-3段）
   - 具体的建议或要求
   - 价值观传递
   - 可操作的方法

5. 落款（个性化）
   - "父字" / "兄字" / 日期 / 地点
```

### 星佳风格特征
- **家学载体**：太爷的日历日记、星佳的文章
- **核心主题**：6个早点、6个国际化、家族使命
- **表达方式**：用故事讲道理，而非说教
- **时间视角**：连接过去（太爷）、现在（自己）、未来（晚辈）

### 参考文章
- 《写给儿子的第一封信》（系列）
- 《2021年做一个无限责任的家族公司》
- 《一封家书》（2025，AI生成验证）

## 五、通用写作检查清单

### 内容层面
- [ ] 有至少一个"信息增量"（读者不知道或没想到的）
- [ ] 有个人经历或案例支撑
- [ ] 避免了纯鸡汤和空话
- [ ] 对读者有实际价值（认知/情感/行动）

### 风格层面
- [ ] 语气真诚，不过度包装
- [ ] 有星佳个人特色（自嘲、真实、具体）
- [ ] 句子长短结合，有节奏感
- [ ] 适当使用口语化表达

### 结构层面
- [ ] 开篇有钩子
- [ ] 段落分明，每段一个核心意思
- [ ] 有金句值得转发
- [ ] 结尾有余味

### 技术层面
- [ ] 标题有足够吸引力但不过度标题党
- [ ] 话题标签准确（#家族传承 #6个早点 等）
- [ ] 配图与内容相关
