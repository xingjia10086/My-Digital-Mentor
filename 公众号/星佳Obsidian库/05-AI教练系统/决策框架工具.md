## 一、6个早点检查清单

### 框架来源
来自星佳2019-2020年的核心思考，在多篇公众号文章中反复强调。

### 6个早点内容

```
1. 早点结婚
2. 早点生娃
3. 早点买房
4. 早点来一线城市
5. 早点做公众号（输出/积累影响力）
6. 早点配置保险
```

### 决策检查表

面对任何重大决策时，问自己：

| 维度 | 检查问题 | 本决策影响 |
|------|----------|------------|
| 结婚 | 这个决策对婚姻/伴侣关系有何影响？ | |
| 生娃 | 对生育计划或现有子女有何影响？ | |
| 买房 | 对房产配置或居住选择有何影响？ | |
| 城市 | 是否需要在一线城市？还是可以其他城市？ | |
| 输出 | 这个决策能否产生内容/影响力积累？ | |
| 保险 | 风险是否可控？是否有对冲手段？ | |

## 三、家族使命对齐检查

### 太爷遗言（家族使命核心）

> "让家里的长辈们去更现代的地方养老，带家里的晚辈们去更文明的地方奋斗！"

### 对齐检查

做任何重大决策时，问自己三个问题：

```
1. 这个决策是否有利于长辈养老？
   - 太奶的医疗/生活质量
   - 父母的养老安排
   - 家族长辈的福祉

2. 这个决策是否有利于晚辈奋斗？
   - 子女的教育机会
   - 晚辈的职业发展
   - 下一代的成长环境

3. 这个决策是否符合"去更文明的地方"？
   - 文明定义：教育水平、法治环境、医疗资源、生活质量
   - 当前选择 vs 其他选择的文明程度对比
```

## 四、决策时间框架

### 星佳的决策节奏

基于历史文章分析，星佳的重大决策遵循以下时间框架：

```
短期（0-6个月）：执行层决策
- 例：具体业务调整、文章选题、客户跟进
- 标准：快速决策，小步试错

中期（1-3年）：战术层决策
- 例：是否换城市、职业转型、子女入学
- 标准：充分调研，果断执行

长期（5-10年）：战略层决策
- 例：家族国际化布局、子女教育路径、资产配置
- 标准：深思熟虑，稳步推进
```

### 当前决策定位

请确定你的决策属于哪个时间框架：

- [ ] 短期（0-6个月）→ 快速决策，减少内耗
- [ ] 中期（1-3年）→ 调研充分，设定里程碑
- [ ] 长期（5-10年）→ 愿景驱动，分步实施

## 五、风险评估矩阵

### 风险维度

| 风险类型 | 评估问题 | 应对策略 |
|----------|----------|----------|
| 财务风险 | 最坏情况下财务是否安全？ | 保留6-12个月应急资金 |
| 健康风险 | 对身体/心理健康的影响？ | 定期体检，保持运动习惯 |
| 关系风险 | 对夫妻关系/家庭关系的影响？ | 充分沟通，共同决策 |
| 机会风险 | 放弃其他选择的代价？ | 明确机会成本，不后悔 |
| 声誉风险 | 对公众形象/个人品牌的影响？ | 保持真实，长期主义 |

## 六、2026决策专项框架

### 背景
星佳面临的2026年核心决策：继续在香港发展 vs 前往海外（新西兰/加拿大/爱尔兰）

### 决策框架

#### Step 1: 明确决策前提
```
- 孟母三迁业务基本盘稳定
- 爱米和小星星可以随行
- 主要考虑因素：教育、医疗、长期发展
```

#### Step 2: 方案对比

| 维度 | 方案A：留港 | 方案B：新西兰 | 方案C：加拿大 | 方案D：爱尔兰 |
|------|------------|--------------|--------------|--------------|
| 6个国际化-身份 | 香港身份✓ | 新西兰PR？ | 加拿大PR？ | 爱尔兰Stamp4？ |
| 6个国际化-教育 | DSE/国际 | 英联邦体系 | 北美体系 | 欧盟体系 |
| 6个国际化-医疗 | 优秀 | 优秀 | 优秀 | 良好 |
| 6个早点-城市 | 一线城市 | 非一线 | 非一线 | 非一线 |
| 6个早点-生娃 | 可冻胚胎 | 可冻胚胎 | 可冻胚胎 | 可冻胚胎 |
| 家族使命 | 符合 | 需评估 | 需评估 | 需评估 |
| 业务延续 | 最优 | 需重建 | 需重建 | 需重建 |
| 语言适应 | 粤语+英语 | 英语 | 英语 | 英语 |
| 社交网络 | 已有基础 | 重建 | 重建 | 重建 |

#### Step 3: 决策触发条件

```
选择方案B/C/D（移居海外）的触发条件：
- [ ] 明确的教育路径优势（对小星星和爱总）
- [ ] 可落地的身份获取方案
- [ ] 业务可远程延续或有新机会
- [ ] 家庭一致同意

选择方案A（留港）的触发条件：
- [ ] 香港教育路径明确可行
- [ ] 孟母三迁业务持续增长
- [ ] 冻胚胎等医疗计划顺利
- [ ] 三胎计划有清晰安排
```

## 八、快速决策流程（日常版）

对于不需要深度思考的日常决策：

```
1. 这个决策符合6个早点吗？
   - 符合 → 继续
   - 不符合 → 重新评估必要性

2. 这个决策对6个国际化有何影响？
   - 正面 → 推进
   - 中性 → 视情况
   - 负面 → 暂缓

3. 最坏结果能否承受？
   - 能 → 推进
   - 不能 → 增加对冲措施

4. 72小时内会有新信息吗？
   - 会 → 等72小时
   - 不会 → 立即决策
```
