#  写作和代码，中年人最应该享受的乐趣！

原创  星佳  [ 星佳是个小人物 ](javascript:void\(0\);)

__ _ _ _ _

都说选择大于努力，之前的人生吃到了公众号的巨大红利，从 14 年陆陆续续写作至今，也算真的把坚持写公众号这件事坚持了下来。

  

于是，终于要开始解决代码的问题，谁能想到这才短短几年，AI
编程的能力的确已经可以淘汰一些初级程序员了，相当于自己最痛苦的事，各种环境配置，解决依赖问题，只能通过 google
和论坛去解决，写代码没写几行，时间全花在解决各种问题上了。

  

昨天写了篇 [ 「给小白写的 Claude Code入门教程」
](https://mp.weixin.qq.com/s?__biz=MzA5ODQwMDgwOA==&mid=2649831369&idx=1&sn=4aceba1926ca7d540add3737a9bca72b&scene=21#wechat_redirect)
，其实发现自己写的还很复杂，最小白的应该是在深圳的美女 AI 博主 @ 秋芝2046 今天发布的视频里教的技巧

  

直接在 cursor 的 IDE 里丢给 Claude Code 的官方文档，然后一句话让 AI 帮你装环境，但可以因为 IP
的原因，只能调用国内的大模型。

  

  

  

但其实看了一遍视频，看起来容易，真要在自己电脑上照着做一遍，你去试试就知道了，可能又会碰到各种报错，环境变量的问题，权限的问题，甚至网络和支付的问题！！

  

又看了下@刘小排 老板写的文章，看到这句话深深共情！

  

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/tOFSAibJ5wqsOibic7uiaBa37eeqpvdiakSrIW4LqEtr2lghYF5BEelFAXKQTpksvdR8ialwfcLvLnBOqwaE1SoNJlKA/640?wx_fmt=jpeg)

  

肉身中国大陆和香港，能流畅的使用 Claude Code 证明自己起码有解决问题的能力了，同样都使用的同一个互联网，但用出来的成果和解决居然差别这么大。

  

甚至每次在使用某个服务之前，得先用一个网站去检测线路和账号有没有问题，像星佳这么笨的人，居然是用一台电脑默认一切都装 C 盘来解决这个问题。

  

这里不得不夸下 Mac 还是好用，让开发者和创作者不用担心各种莫名其妙的环境和配置问题，安心在创作中，包括代码和写作！

  

没有复用的知识的确不需要学习了，现在直接把报错信息和截图丢给 GPT，Cursor 等等，它就会告诉你应该如何处理，之前用 GPT Plus
的问题在于只能给你出图文并茂的答案，这两天试着用 Cursor，已经直接可以帮你下载，配置和启动很多服务了。

  

命令行和图形化界面，似乎我们都应该用起来，取长补短，以实现目标为主，就跟之前文章里星佳提到的观点，从来不参与什么 win 好还是 Mac 好，iPhone
好还是安卓好之类的，成年人，都买来都用，配合着用，达到自己的目标即可，取长补短才是！

  

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/tOFSAibJ5wqsOibic7uiaBa37eeqpvdiakSrI5kGMA9GibiaXoY7aHDRRQialnL3IyCoth3A6whTo1vyC0EibsUhBc98qAQ/640?wx_fmt=jpeg)

  

真的越来越感叹，运气好生在这样的时代，通过互联网可以学习大部分你想学习的知识，而且人和人之间不需要见面就可以付款给对方，居然你其实和世界首富用的生产力工具都是同一个品牌的，还能追踪人家每天的动态，公司的业绩等等。

  

这哪怕放到 10 年前，似乎也是遥不可及的吧。

  

所以，人到中年，把爱好兴趣换成写作和代码，会发现必须独处才能安心下来，躲在书房，有网线和电脑，就拥有了全世界吧。

  

  

如果一个男的不抽烟不喝酒不赌博也没驾照，对相机毫无兴趣，似乎过的真的苦行僧生活？

  

摘录一段知识星球里问 老猫对于 李笑来的看法，期待之后猫总写的文章。

  

优秀的人，的确是扎堆的，有圈子的。

  

前年偶然间知道李笑来老师做了一个面向家长的社群，机缘巧合认识某个一起经营那个付费社群的人，于是一次性买了好多年的会员。

  

原本打算进去「人生最重要一课」的社群，但是关闭了入群通道，于是耐心学了点他的社群里持续更新的课程，还有时不时李老师语音的心理按摩。

  

真的不得不感叹李老师太厉害了，学啥都能写出教程，而且目前他的生活状态是我知道的人里我的确最羡慕的，养活那么多娃，和老婆从不吵架，默默做公益在西部植树好多年，还在得到开课，各种出书，社群也经营的很好，又在香港合规经营自己的基金「他说是就给孩子们的礼物」，又听说正在日本买了块地盖自己的学校，要自己教自己的孩子，而不是丢给学校。

  

所以，作为和李笑来真的一起工作过，创业过的猫总，想从你的视角听听关于李老师的故事。

  

怎么可能会有一个销售出身的人，能把英语，写作和编程，赚钱等等能力都做到如此水平。

  

还经营父母社群，自己亲自写书教学自己的孩子们，最近又把烟戒了，年过 50 身材也保持的很好，真是值得很多父亲学习的榜样。

  

虽然区块链时代也有很多烂新闻，但对这个人真的很好奇。
