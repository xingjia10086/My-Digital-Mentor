换个城市，旅居一阵子？

  

  

老读者们都知道星佳一直有个 6 个早点和国际化的愿景，而且一直朝着终极目标努力，而其中「  两本护照+海  外  PR  」的目标已经快完成进度的
60%，剩下的无非就是熬时间静待花开就好。

  

说起海外 PR，为何会选择日本？  

  

先用排除法，值得中国人拿绿卡的国家其实一只手就能数过来。

  

先说说美国，一方面中美关系的紧张，除此之外EB5需要很久的排期和投资， EB1A 又不符合要求，哪怕去读书抽签也是看命，及时拿到了 PR 依然要有维持 PR
的时间成本，每年需要居住半年以上，并且全球征税和遗产税等等，所以美国PR不适合我们的情况，况且目前的确也没那个实力去拿。  

  

隔壁的加拿大很容易，其实只要肉身能过去，不管读书还是工作，甚至投资拿个枫叶卡都不难，至于为何不适合的理由跟美国差不多，维护 PR 的成本太高，每年需要待
180 以上才能保住枫叶卡。

  

虽然有不少人真的过着国内住半年，加拿大住半年的生活，除此之外还有某些放得下的夫妻配合双打，一人入籍+带娃定居加拿大从小国际教育，而另一人自由身份常住国内赚钱。

  

澳洲也是很多人的选择，理由跟如上差不多，只能二选一，不能既要又要。  

  

身边也有土豪选择了爱尔兰，不过人家赶在政策结束之前几十万欧元的投资，最香的是维持绿卡每年只需要登录一天即可，希望土豪们最后真的如愿，毕竟有钱人的确会在很多事上永远有
Plan B。  

  

所以，在惊涛骇浪来之前多个 PR
的选择题最后剩下的答案，只能在亚洲国家选，刚好想什么来什么，所谓的孕妇效应，去年终于认识了好几个已经在日本定居的网友，毕竟看了很多年的公众号，于是见面后跟星佳聊了很多日本，尤其从他们嘴里听到的日本生活，就跟当年还没来深圳之前那些深圳人给我讲的深圳一样。  

  

反倒从日本留学了四年的表弟那里，得到了大部分都是针对日本的负面评价，倒也理解，任何一个城市，穷人或许都过不上理想的生活，况且他还要靠父母资助学业的穷留学生。  

  

但有一点他回深圳后一阵子完全接受不了的现状，怎么工作按照小时工资核算居然会这么低，而且加班从来不会有人跟你提加班工资这回事呢？

  

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/tOFSAibJ5wqt5q6FzaYNssEIyKl3RIiaZ15uDqoz0BhnJanoUjPY85LnicEpSvNpDFBjGFTia74NC93XMzRqGsdNZw/640?wx_fmt=jpeg)

  

  

但星佳对日本的印象还停留在 18 年的旅行中，那时候在游客的视角下觉得跟大城市没啥两样，但偏偏城市的魅力和文明程度只能听听长期生活在那边的人怎么说。

  

然后今年很多热门新闻就是巴菲特和索罗斯等等金融大鳄也用行动重仓了日本，疫情后又成了全球热门旅游地，于是民宿和酒店爆满。  

  

研究了一阵子，惊奇的发现很多上海和香港人最喜欢去的地方也是日本，的确去趟日本飞机也就深圳回银川差不多的时间，加上文化和饮食习惯类似，很适合去旅居感受下不同文化的生活。  

  

至于为何想拿日本PR，还有一个考虑是关于养老，除了深圳和香港，一直在规划第三个适合养老的城市，日本和香港的人均寿命 87
岁就是最好的指标，医疗水平，饮食健康和自然环境等等。  

  

有些人一旦开始心里有了种子，开始琢磨某个事，那就一定要整出个结果再罢休，于是安排两个定居在日创业的朋友给孟母三迁的小伙伴们做了个内部分享，详细的讲了讲为何他们会选择拖家带口定居日本，并且一定要拿到
PR 的原因？  

  

偷出来几张 PPT 给大家看看，希望能给迷茫的中产或许给一个选择答案。  

  

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/tOFSAibJ5wqt5q6FzaYNssEIyKl3RIiaZ1twcaLoibfbX2ABVmA5BhTliasW2vxxPUMoafSL8cyXNhGKnrg13RMfCQ/640?wx_fmt=jpeg)

  

因为熟悉香港的落户政策，特地把日本 PR
的政策和港府的对比下，全球都在抢人才，其实手段都差不多，要么你有钱，要么你有才华，要么你有时间去做服务业，跟香港身份的政策比，日本其实更容易，至于详情今天就不说了，很多东西本身就是难者不会会者不难吧。  

  

但跟港府只认可第一学历不同，在日本那边居然认可本科，研究生和博士的 qs100 学历，所谓 J-Find，而 J-Skip
给了很多类似外交官的特权，用实实在在的政策去吸引有钱人，比如有钱人可以合法带佣人过去，配合香港身份搭配下的菲佣，又是一种 Match 的玩法。

  

至于其他的，等星佳再研究研究开个付费直播再聊吧，毕竟可想而知，骂日本的应该比骂香港的还多，这就是互联网的现状。

  

没有机会去看过外面的世界，因为外面太危险。
